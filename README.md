# knockpy-gui
Knockpy GUI

# Install

`git clone https://github.com/guelfoweb/knockpy-gui.git`

`cd knockpy-gui`

`virtualenv --python=python3 venv3`

`source venv3/bin/activate`

`pip3 install -r requirements.txt`

# Start

`python3 manage.py runserver`

then visit: `http://127.0.0.1:8000/`

user: `admin` 
pass: `admin`
